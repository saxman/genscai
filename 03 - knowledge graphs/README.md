# Knowledge Graphs

Build and query knowledge graphs over the literature — a GraphRAG pipeline and a LangChain LLM-graph-transformer approach — to surface entities and relationships rather than ranked passages.

Part of the [genscai](../README.md) use-case series. Shared library code lives in the [`genscai`](../genscai) package; datasets in [`data/`](../data) and generated artifacts in `output/` are shared at the repo root.
